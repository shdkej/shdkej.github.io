  • unix, linux = monolithic. base on kernel
  • micro kernel is base on each process. and communicate with other messenger
  • micro kernel can use general hardware. linux made for 1 target hardware. it is good to use more hardware performance. but it depends on 1 hardware. micro kernel's problem is don't use hardware performance efficiently. micro kernel has portability.
